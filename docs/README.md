# Math formulas
## Area
- Circle: S = πR²
- Rectangle: S = ab
- Square: S = a²

## Perimeter
- Circle: P = 2πR
- Rectangle: P = 2a + 2b
- Square: P = 4a

# Area functions
## Circle
- На вход принимает число r, возвращает квадрат числа r, умноженный на число pi
    - Пример:
        - Ввод:
        - r = 3
        - Вывод:
        - area = 28.26

## Square
- На вход принимает число a, возвращает квадрат числа a
    - Пример:
        - Ввод:
        - a = 3
        - Вывод:
        - area = 9

## Rectangle
- На вход принимает число a и число b, возвращает произведение a и b
    - Пример:
        - Ввод:
        - a = 3
        - b = 4
        - Вывод:
        - area = 12

## Triangle
- На вход принимает число a и число h, возвращает произведение чисел a и h, делённое на 2
    - Пример:
        - Ввод:
        - a = 2
        - h = 2
        - Вывод:
        - area = 2

# Perimeter functions
## Circle
- На вход принимает число r, возвращает удвоенное число r, умноженное на число pi
    - Пример:
        - Ввод:
        - r = 3
        - Вывод:
        - perimeter = 18.84

## Square
- На вход принимает число a, возвращает произведение a * 4
    - Пример:
        - Ввод:
        - r = 3
        - Вывод:
        - perimeter = 12

## Rectangle
- На вход принимает число a и число b, возвращает удвоенную сумму a и b
    - Пример:
        - Ввод:
        - a = 3
        - b = 4
        - Вывод:
        - perimeter = 14

## Triangle
- На вход принимает число a, число b и число c, возвращает сумму чисел a, b и c
    - Пример:
        - Ввод:
        - a = 3
        - b = 4
        - c = 5
        - Вывод:
        - perimeter = 12
